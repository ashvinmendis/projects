Smart Glove

Achived as best group projeect and won the gold medal

This is the final year group project for an MEng Mechatronics Engineering degree. I was responsible for the MCU programming and circuit design of the smart glove. The circuit was designed using Eagle CAD software, and the programming was done using Arduino. The smart glove is designed to assist deaf individuals in communicating with others. When wearing this glove and moving their hand, it converts sign language into voice output. A mobile phone is utilized to provide the voice output.

The smart glove and the mobile phone are connected via Bluetooth. Flex sensors and gyroscopic sensors are used to capture the positions and angles of fingers and the hand. By processing this data through the microcontroller, the final data values are sent to the mobile application via Bluetooth. The mobile application can then search for specific data values to generate voice output.
